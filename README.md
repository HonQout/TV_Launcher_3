# TV Launcher 3
## Basic information
This is a launcher application designed for Android TV by using Kotlin and Jetpack Compose.
## Last updated: 2025-11-09
### Changes and improvements:
1. Finished InputScreen.kt. Now this app is able to display screen of input. 
<Notice> This function has not been tested due to lack of device.
## Function
1. Fix your favourite applications on home screen so that you can access them easily.
2. Displays all launchable applications on your device and provides entrances to manage them.
3. Offer quick access to commonly used system settings.
