# TV Launcher 3
## Basic information
This is a launcher application designed for Android TV by using Kotlin and Jetpack Compose.
## Last updated: 2025-11-13
### Fixed:
1. **Critical** The issue that the shortcut of an app on HomeScreen will not be removed or updated if it is uninstalled or updated.
2. **Critical** The list of resolveInfos will not re-sort when system locale is changed.
## Function
1. Direct access to your favourite applications on home screen by simply fix them on it.
2. Displays all launchable applications on your device and provides entrances to manage them.
3. **Experimental** Displays screen of input sources.
4. Quick access to commonly used system settings.
